## [1.0.0] - 2026-02-09

### Added
- Initial release of ShimmerKit
- `ShimmerModifier` for applying shimmer animation to views
- `ShimmerIfLoading` for conditional shimmer based on loading state
- `.shimmer()` view extension for easy application
- `.shimmer(isLoading:)` view extension for conditional shimmer
- Demo app showcasing feed-style skeleton screens
- Comprehensive documentation and usage examples
- Swift Package Manager support
- iOS 15+ and macOS 12+ support
